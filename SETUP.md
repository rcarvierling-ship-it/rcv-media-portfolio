# RCV.Media Setup Guide

This file explains how to set up the RCV.Media project locally.

## Project Overview

RCV.Media is a photography portfolio, booking, and client gallery website for Reese Vierling.

The brand offers:

- Seniors
- Portraits
- Sports
- Events
- Graduation / cap & gown
- Team media days

Main positioning:

> Portraits, seniors, sports, and events — captured with energy and style.

---

## Tech Stack

- Next.js
- React
- Tailwind CSS
- Framer Motion
- Supabase
- Cloudinary
- Vercel
- GitHub
