# RCV.Media Project Rules

RCV.Media is a photography/media brand by Reese Vierling.

The brand offers:
- Seniors
- Portraits
- Sports
- Events
- Graduation / cap & gown
- Team media days

Do not make the brand only sports-focused.

Main positioning:
Portraits, seniors, sports, and events — captured with energy and style.

Style:
- Premium
- Dark
- Cinematic
- Image-first
- Mobile-first
- Clean
- Not hacker/terminal themed

Do not use video-related wording:
- no videography
- no highlight reels
- no video clips
- no reel clips
- no video packages

Tech stack:
- Next.js
- Supabase
- Cloudinary
- Tailwind CSS
- Framer Motion

Images:
- Store uploaded images in Cloudinary
- Store image metadata in Supabase
- Do not store images locally
- Do not store Base64 images in the database

Dashboard:
- Photo upload/edit/delete
- Albums/client galleries
- Bookings pipeline
- Pricing package editor
- Site settings editor
- Inquiries inbox
- Private gallery passcodes

Never expose API secrets in client-side code.
Never edit .env.local unless explicitly asked.
