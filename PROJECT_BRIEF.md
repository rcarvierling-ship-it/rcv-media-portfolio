# RCV.Media Project Brief

RCV.Media is Reese Vierling's photography portfolio and booking website.

Goal:
Create a professional photography brand website for seniors, portraits, sports, events, graduation sessions, and team media days.

Main pages:
- Home
- Portfolio
- Services
- Pricing
- Book
- Client Galleries
- About
- Contact

The site should feel premium, cinematic, bold, and image-first.

The dashboard should allow Reese to manage:
- Photos
- Galleries
- Bookings
- Pricing
- Site settings
- Inquiries
